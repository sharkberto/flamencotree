YouTube iframe API for Video Control

SIMPLE
1. Load video
2. User sets breakpoints and adds metadata to describe section
3. Video information saved and displayed in page

SIMPLE 2 (uses OAuth access)
1. Load video based on user query
2. User labels a clip of the video, by setting the breakpoints, controls (playback quality) and metadata for section
3. Information saved somewhere, to user's account
4. Next time video is loaded in webpage with access to user playlist, settings are saved
